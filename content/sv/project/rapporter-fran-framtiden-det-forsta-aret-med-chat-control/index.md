Extrapolerande fiktion av Jeremiah Lee som föreställer sig rubriker från det första året efter att EU inför massövervakning av privata meddelanden.

<!--more-->

[Rapporter från framtiden: Det första året med Chat Control](https://www.jeremiahlee.com/posts/after-chat-control/sv/) är en tankeövning skriven, designad och kodad av [Jeremiah Lee](https://www.jeremiahlee.com/). Texten föreställer sig konsekvenserna om Europeiska unionen antog ett förslag om att skanna foton och videor som människor skickar för att upptäcka eventuella övergrepp mot barn. Förslaget är verkligt, men artikeln är extrapolerande fiktion byggd på verkliga uttalanden i nya sammanhang.

Mattias Axell bidrog till projektet med feedback på den svenska översättningen och genom att översätta texten från engelska till svenska. Den engelska originaltexten finns som [Dispatches from the future: The first year of Chat Control](https://www.jeremiahlee.com/posts/after-chat-control/).
